## Get Started

DALLE POLAROID is a tool for using AI model to generate a polaroid style photo from a text description. 

### Install

Execute the following commands to download the source code

```
git clone https://github.com/Kiotlin/dalle-polaroid.git
cd dalle-polaroid
```

Execute the following commands to install dependencies

```
npm install
```

### Launch

```
npm start
```
